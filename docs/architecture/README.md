# IPPAN-L2 Architecture

This directory contains the architectural definitions for the IPPAN-L2 node and its interactions with IPPAN CORE (L1).

*   **[Layer Architecture](./layer-architecture.md)**: High-level split between L1 Settlement and L2 Execution.
*   **[Hub Registry](./hub-registry.md)**: Definitions of the canonical Hubs (FIN, DATA, etc.).
*   **[Hubs](./hubs.md)**: `HubStateMachine` and internal architecture.
*   **[Settlement Contract](./settlement-contract.md)**: The types and interfaces for L2->L1 communication.
*   **[Settlement Lifecycle](./settlement-lifecycle.md)**: Detailed transaction flow from L2 submission to L1 finality.
*   **[Economics](./economics.md)**: Fee models and integer arithmetic standards.
